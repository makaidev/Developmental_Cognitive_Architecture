# MAKAIDEV — A Developmental Hypothesis of General Intelligence

> Intelligence is not a larger model. It is a developmental environment where intelligence
> emerges over time through structured interaction, memory, and feedback.

This repository contains an open research note proposing that general intelligence is a
**developmental-system problem**, not a scaling problem. It outlines the **Personal Cognitive OS**
architecture (Vault → Ontology → Mutation → Dream → Simulation → Reality → Feedback) and a
seven-level, growth-based evaluation framework.

📄 **Read the paper:** [`MAKAIDEV_AGI_Hypothesis_Paper.md`](./MAKAIDEV_AGI_Hypothesis_Paper.md)

## What this is

- A **hypothesis** offered for critique and cross-validation.
- A **research manifesto** for developmental AI and ontology-based memory.

## What this is not

- A claim of having built AGI.
- A finished theory or a product.

## Core ideas

| Idea | One line |
|------|----------|
| Immutable Vault | Original memory is never edited in place — only clones are mutated. |
| Ontology Graph | Knowledge as a weighted, timestamped relational graph. |
| Mutation Engine | Creativity = existing concepts + a controlled variation rate. |
| Sensory Abstraction | Model the *decision-influence* of senses, not sensation itself. |
| Sleep Cycle | Offline consolidation enables safe continual learning. |
| 7-Level Evaluation | Judge growth, persistence, and self-generated goals — not benchmarks. |
